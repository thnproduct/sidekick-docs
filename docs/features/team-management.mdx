## Roles

### Admins
- Repository management (add / delete repositories to the organization)
- Rule management (add / delete rules)
- Organization management ( add / delete users, change roles)
- Billing management (changing plan, adding / changing credit card information - not-applicable for open source projects)
- Integration management (managing github and slack integration settings)
- Everything user role has

### Users
- Read-only access to workflow / test run details
- Read-only access to highlights
