# Slack

With the help of the [Incoming WebHooks](https://slack.com/apps/A0F7XDUAZ-incoming-webhooks) Foresight now supports Slack!

You can generate a WebHook url and get notified when your workflows and tests runs has finished. You can add up to 10 webhooks.
